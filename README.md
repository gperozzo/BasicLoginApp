# BasicLoginApp
Basic login app in Xamarin.Android with local data repository through `Json` files.

<h2>Features</h2>

- User registratin with username and password;
- Sign up with users registered.

<h2>Built with</h2>

- [Visual Studio](https://visualstudio.microsoft.com/)
